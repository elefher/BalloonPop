# Baby Pop
This is a simple Phaser 3 game I made for two reasons:
- I wanted some experience with Phaser 3
- I wanted to make a game for my 2 year old daughter

There's not many Phaser 3 tutorials out there (as of March 2018), so I decided to make this public in the hopes it would help somebody.
